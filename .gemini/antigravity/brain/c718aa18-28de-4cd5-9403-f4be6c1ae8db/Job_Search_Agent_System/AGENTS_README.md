# Coordination entre agents IA

> Point d'entrée pour tout agent IA travaillant sur ce projet.

## Fichiers à consulter

| Fichier | Usage |
|---------|-------|
| **[AGENTS_ROADMAP.md](AGENTS_ROADMAP.md)** | Architecture, règles, workflow — **lire en premier** |
| **[AGENTS_LOG.md](AGENTS_LOG.md)** | Journal des actions réalisées — historique et handoff |
| **[AGENTS_TODO.md](AGENTS_TODO.md)** | Tâches en attente — phases 2, 3, 4 et backlog |

## Workflow rapide

1. Lire `AGENTS_ROADMAP.md`
2. Consulter `AGENTS_LOG.md` (dernières actions)
3. Choisir une tâche dans `AGENTS_TODO.md`
4. Après action : mettre à jour `AGENTS_LOG.md` + `AGENTS_TODO.md`
